[2] Linked List Cycle

题目

  • 概述
    是否有圈。

  • 思路
    用两个指针，一个快一个慢，看二者是否相遇。（快指针走2，慢指针走1，在M后，二者会相遇）
    若没有圈，快指针会先走到NULL

[3] Linked List Cycle II

[题目]（https://leetcode.com/explore/learn/card/linked-list/214/two-pointer-technique/1214/）

  • 概述
    找出圈开始的位置（最后一个元素指向的元素）

  • 思路
    存下所有点，重复的第一个就是圈开始的位置。
