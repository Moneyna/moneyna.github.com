关于多个c++文件的makefile编写练习

发表于 2020-04-11 分类于 C++

1 背景

在为自己的纯前端项目生成测试数据。
makefile部分正确无误，然而面临的真实困境却是生成的json数据格式有问题……有问题的本质原因还是Vue需要的就是这个有问题格式。
sigh

2 makefile文件及注意事项

需要注意:

  • 第一个间隔是TAB而不是空格

  • .o文件的名字必须与文件名一致。如果不一致会报错文件不存在（实际上也的确不存在就是）

  • -o参数的书写位置，必须在所有.o文件只后，在最终运行文件前。否则报main函数不存在

3 c++相关问题

因为本工程结构涉及三个c++文件对同一个json文件操作，一定要注意即时关闭及设置文件操作格式为追加
